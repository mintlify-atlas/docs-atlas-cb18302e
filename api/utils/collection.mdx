## Collection

A collection class that extends the built-in `Map` with array-like methods for easier data manipulation.

### Type Definition

```typescript
class Collection<K, V> extends Map<K, V>
```

### Type Parameters

- `K` - The type of keys in the collection
- `V` - The type of values in the collection

### Methods

#### sweep

Removes elements from the collection based on a filter function.

```typescript
sweep(fn: (value: V, key: K, collection: this) => unknown): number
```

**Parameters:**
- `fn` - The filter function that determines which elements to remove

**Returns:** The number of elements removed from the collection

**Example:**
```typescript
const collection = new Collection<number, string>();
collection.set(1, 'one');
collection.set(2, 'two');
collection.set(3, 'three');
const removedCount = collection.sweep((value, key) => key % 2 === 0);
console.log(removedCount); // Output: 1
console.log(collection.size); // Output: 2
```

#### map

Creates a new array with the results of calling a provided function on every element in the collection.

```typescript
map<T = any>(fn: (value: V, key: K, collection: this) => T): T[]
```

**Parameters:**
- `fn` - The function that produces an element of the new array

**Returns:** A new array with the results

**Example:**
```typescript
const collection = new Collection<number, string>();
collection.set(1, 'one');
collection.set(2, 'two');
collection.set(3, 'three');
const mappedArray = collection.map((value, key) => `${key}: ${value}`);
console.log(mappedArray); // Output: ['1: one', '2: two', '3: three']
```

#### filter

Creates a new array with all elements that pass the test implemented by the provided function.

```typescript
filter(fn: (value: V, key: K, collection: this) => boolean): V[]
```

**Parameters:**
- `fn` - The function to test each element of the collection

**Returns:** A new array with the elements that pass the test

**Example:**
```typescript
const collection = new Collection<number, string>();
collection.set(1, 'one');
collection.set(2, 'two');
collection.set(3, 'three');
const filteredArray = collection.filter((value, key) => key % 2 === 0);
console.log(filteredArray); // Output: ['two']
```

#### reduce

Applies a function against an accumulator and each element in the collection to reduce it to a single value.

```typescript
reduce<T = any>(
  fn: (accumulator: T, value: V, key: K, collection: this) => T,
  initialValue?: T
): T
```

**Parameters:**
- `fn` - The function to execute on each element
- `initialValue` - The initial value of the accumulator

**Returns:** The value that results from the reduction

**Example:**
```typescript
const collection = new Collection<number, number>();
collection.set(1, 1);
collection.set(2, 2);
collection.set(3, 3);
const sum = collection.reduce((acc, value) => acc + value, 0);
console.log(sum); // Output: 6
```

#### every

Checks if all elements in the collection pass a test implemented by the provided function.

```typescript
every(fn: (value: V, key: K, collection: this) => boolean): boolean
```

**Parameters:**
- `fn` - The function to test each element

**Returns:** `true` if all elements pass the test, otherwise `false`

**Example:**
```typescript
const collection = new Collection<number, number>();
collection.set(1, 1);
collection.set(2, 2);
collection.set(3, 3);
const allGreaterThanZero = collection.every(value => value > 0);
console.log(allGreaterThanZero); // Output: true
```

#### some

Checks if at least one element in the collection passes a test implemented by the provided function.

```typescript
some(fn: (value: V, key: K, collection: this) => boolean): boolean
```

**Parameters:**
- `fn` - The function to test each element

**Returns:** `true` if at least one element passes the test, otherwise `false`

**Example:**
```typescript
const collection = new Collection<number, number>();
collection.set(1, 1);
collection.set(2, 2);
collection.set(3, 3);
const hasEvenValue = collection.some(value => value % 2 === 0);
console.log(hasEvenValue); // Output: true
```

#### find

Returns the value of the first element in the collection that satisfies the provided testing function.

```typescript
find(fn: (value: V, key: K, collection: this) => boolean): V | undefined
```

**Parameters:**
- `fn` - The function to test each element

**Returns:** The value of the first element that passes the test, or `undefined`

**Example:**
```typescript
const collection = new Collection<number, number>();
collection.set(1, 1);
collection.set(2, 2);
collection.set(3, 3);
const firstEvenValue = collection.find(value => value % 2 === 0);
console.log(firstEvenValue); // Output: 2
```

#### findKey

Returns the first key in the collection that satisfies the provided testing function.

```typescript
findKey(fn: (value: V, key: K, collection: this) => boolean): K | undefined
```

**Parameters:**
- `fn` - The function to test each element

**Returns:** The first key that passes the test, or `undefined`

**Example:**
```typescript
const collection = new Collection<number, number>();
collection.set(1, 1);
collection.set(2, 2);
collection.set(3, 3);
const firstEvenKey = collection.findKey(value => value % 2 === 0);
console.log(firstEvenKey); // Output: 2
```

## LimitedCollection

A collection with built-in size limits and automatic expiration for elements.

### Type Definition

```typescript
class LimitedCollection<K, V>
```

### Constructor Options

```typescript
interface LimitedCollectionOptions<K, V> {
  limit: number;
  expire: number;
  onDelete?: (key: K, value: V) => void;
  resetOnDemand: boolean;
}
```

**Options:**
- `limit` - Maximum number of elements in the collection (default: `Infinity`)
- `expire` - Expiration time in milliseconds (default: `0` - no expiration)
- `onDelete` - Callback function called when an element is deleted
- `resetOnDemand` - Whether to reset expiration timer on access (default: `false`)

### Properties

#### size

Returns the number of elements in the limited collection.

```typescript
get size(): number
```

#### closer

Returns the element in the collection that is closest to expiration.

```typescript
get closer(): LimitedCollectionData<V> | undefined
```

### Methods

#### set

Adds an element to the limited collection.

```typescript
set(key: K, value: V, customExpire?: number): void
```

**Parameters:**
- `key` - The key of the element
- `value` - The value of the element
- `customExpire` - Custom expiration time for this specific element (optional)

**Example:**
```typescript
const collection = new LimitedCollection<number, string>({ limit: 3 });
collection.set(1, 'one');
collection.set(2, 'two');
collection.set(3, 'three');
console.log(collection.size); // Output: 3
collection.set(4, 'four');
console.log(collection.size); // Output: 3
console.log(collection.get(1)); // Output: undefined
```

#### get

Returns the value of an element in the limited collection.

```typescript
get(key: K): V | undefined
```

**Parameters:**
- `key` - The key of the element

**Returns:** The value of the element, or `undefined` if not found

**Example:**
```typescript
const collection = new LimitedCollection<number, string>();
collection.set(1, 'one');
const value = collection.get(1);
console.log(value); // Output: 'one'
```

#### has

Checks if an element exists in the limited collection.

```typescript
has(key: K): boolean
```

**Parameters:**
- `key` - The key of the element

**Returns:** `true` if the element exists, `false` otherwise

**Example:**
```typescript
const collection = new LimitedCollection<number, string>();
collection.set(1, 'one');
console.log(collection.has(1)); // Output: true
console.log(collection.has(2)); // Output: false
```

#### delete

Removes an element from the limited collection.

```typescript
delete(key: K): boolean
```

**Parameters:**
- `key` - The key of the element to remove

**Returns:** `true` if the element was removed, `false` otherwise

**Example:**
```typescript
const collection = new LimitedCollection<number, string>();
collection.set(1, 'one');
console.log(collection.delete(1)); // Output: true
console.log(collection.delete(2)); // Output: false
```

#### raw

Returns the raw data of an element, including expiration information.

```typescript
raw(key: K): LimitedCollectionData<V> | undefined
```

**Parameters:**
- `key` - The key of the element

**Returns:** The raw data object or `undefined`

**Example:**
```typescript
const collection = new LimitedCollection<number, string>();
collection.set(1, 'one');
const rawData = collection.raw(1);
console.log(rawData); // Output: { value: 'one', expire: -1, expireOn: -1 }
```

#### clear

Removes all elements from the collection.

```typescript
clear(): void
```

#### keys

Returns an iterator of keys in the collection.

```typescript
keys(): IterableIterator<K>
```

#### values

Returns an iterator of values in the collection.

```typescript
values(): IterableIterator<LimitedCollectionData<V>>
```

#### entries

Returns an iterator of key-value pairs in the collection.

```typescript
entries(): IterableIterator<[K, LimitedCollectionData<V>]>
```

### Usage Example

```typescript
import { LimitedCollection } from 'seyfert';

// Create a cache with a 5-second expiration
const cache = new LimitedCollection<string, User>({
  limit: 100,
  expire: 5000,
  onDelete: (key, value) => {
    console.log(`User ${value.name} removed from cache`);
  },
  resetOnDemand: true // Reset expiration when accessed
});

// Add users to cache
cache.set('user1', { id: '1', name: 'Alice' });
cache.set('user2', { id: '2', name: 'Bob' });

// Access will reset expiration if resetOnDemand is true
const user = cache.get('user1');
```
