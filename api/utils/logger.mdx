## Logger

A flexible logging utility for managing log messages with various severity levels and optional file persistence.

### Type Definition

```typescript
class Logger
```

### Constructor

```typescript
new Logger(options: LoggerOptions)
```

**Options:**
```typescript
interface LoggerOptions {
  logLevel?: LogLevels;
  name?: string;
  active?: boolean;
  saveOnFile?: boolean;
}
```

- `logLevel` - Minimum log level to display (default: `LogLevels.Debug`)
- `name` - Name of the logger instance (default: `'seyfert'`)
- `active` - Whether the logger is active (default: `true`)
- `saveOnFile` - Whether to save logs to a file (default: `false`)

### Log Levels

```typescript
enum LogLevels {
  Debug = 0,
  Info = 1,
  Warn = 2,
  Error = 3,
  Fatal = 4,
}
```

Log levels determine which messages are displayed. Messages with a level lower than the configured `logLevel` will be ignored.

### Usage Example

```typescript
import { Logger, LogLevels } from 'seyfert';

// Create a logger instance
const logger = new Logger({
  name: 'MyBot',
  logLevel: LogLevels.Info,
  saveOnFile: true
});

// Log different severity levels
logger.debug('This will not be shown (below Info level)');
logger.info('Bot is starting...');
logger.warn('Configuration file is missing, using defaults');
logger.error('Failed to load plugin', error);

// Change log level dynamically
logger.level = LogLevels.Debug;
logger.debug('Now debug messages will show');

// Disable logging temporarily
logger.active = false;
logger.info('This will not be logged');
logger.active = true;
```

### Customization Example

```typescript
import { Logger, LogLevels } from 'seyfert';

// Customize log format
Logger.customize((logger, level, args) => {
  const levelName = LogLevels[level];
  const timestamp = new Date().toISOString();
  return [
    `[${timestamp}]`,
    `[${levelName}]`,
    `[${logger.name}]`,
    ...args
  ];
});

// Customize filename
Logger.customizeFilename((logger) => {
  const date = new Date().toISOString().split('T')[0];
  return `${logger.name}-${date}.log`;
});

// Enable file logging for all loggers
Logger.saveOnFile = 'all';

const logger = new Logger({ name: 'MyBot' });
logger.info('This will be logged with custom format');
```
