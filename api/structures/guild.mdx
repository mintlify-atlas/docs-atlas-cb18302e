The `Guild` class represents a Discord guild (server) with all its properties and methods for managing guild resources.

## Constructor

```typescript
new Guild(client: UsingClient, data: APIGuild | GatewayGuildCreateDispatchData)
```

## Properties

### Core Properties

Inherits all properties from `APIGuild` in camelCase format, including:

- `id: string` - The guild's unique identifier
- `name: string` - The guild's name
- `icon: string | null` - The guild's icon hash
- `ownerId: string` - The user ID of the guild owner
- `permissions?: string` - Total permissions for the user in the guild
- `region?: string` - Voice region
- `afkChannelId: string | null` - AFK channel ID
- `afkTimeout: number` - AFK timeout in seconds
- `verificationLevel: number` - Verification level required
- `defaultMessageNotifications: number` - Default message notification level
- `explicitContentFilter: number` - Explicit content filter level
- `features: string[]` - Enabled guild features
- `mfaLevel: number` - Required MFA level
- `applicationId: string | null` - Application ID if bot-created
- `systemChannelId: string | null` - System messages channel ID
- `systemChannelFlags: number` - System channel flags
- `rulesChannelId: string | null` - Rules channel ID
- `maxPresences?: number | null` - Maximum presences
- `maxMembers?: number` - Maximum members
- `vanityUrlCode: string | null` - Vanity URL code
- `description: string | null` - Guild description
- `banner: string | null` - Banner hash
- `premiumTier: number` - Premium tier (boost level)
- `premiumSubscriptionCount?: number` - Number of boosts
- `preferredLocale: string` - Preferred locale
- `publicUpdatesChannelId: string | null` - Public updates channel ID
- `maxVideoChannelUsers?: number` - Max video channel users
- `welcomeScreen?: WelcomeScreen` - Welcome screen configuration
- `nsfwLevel: number` - NSFW level
- `premiumProgressBarEnabled: boolean` - Whether premium progress bar is enabled

### Gateway-specific Properties

These properties are only available when the guild is received through gateway events:

- `joinedAt: number` - Timestamp when bot joined the guild
- `memberCount: number` - Total member count
- `large: boolean` - Whether the guild is considered large
- `unavailable?: boolean` - Whether the guild is unavailable

### Computed Properties

#### maxStickers

```typescript
get maxStickers(): MaxStickers
```

Returns the maximum number of custom stickers allowed based on premium tier:
- Tier 0: 5 stickers
- Tier 1: 15 stickers
- Tier 2: 30 stickers
- Tier 3: 60 stickers

**Returns:** `5 | 15 | 30 | 60`

#### maxEmojis

```typescript
get maxEmojis(): MaxEmojis
```

Returns the maximum number of custom emojis allowed based on premium tier:
- Tier 0: 50 emojis
- Tier 1: 100 emojis
- Tier 2: 150 emojis
- Tier 3: 250 emojis

**Returns:** `50 | 100 | 150 | 250`

## Methods

### fetchOwner

```typescript
fetchOwner(force?: boolean): Promise<GuildMember | null>
```

Fetches the guild owner as a guild member.

**Parameters:**
- `force` - Whether to skip cache and fetch from API

**Returns:** Promise resolving to the guild owner member, or null if no owner exists

**Example:**
```typescript
const owner = await guild.fetchOwner();
if (owner) {
  console.log(`Guild owner: ${owner.user.username}`);
}
```

### listActiveThreads

```typescript
listActiveThreads(force?: boolean): Promise<ThreadChannel[]>
```

Lists all active threads in the guild.

**Parameters:**
- `force` - Whether to skip cache and fetch from API

**Returns:** Promise resolving to array of active thread channels

**Example:**
```typescript
const threads = await guild.listActiveThreads();
console.log(`Active threads: ${threads.length}`);
```

### edit

```typescript
edit(body: RESTPatchAPIGuildJSONBody, reason?: string): Promise<Guild>
```

Edits the guild's settings.

**Parameters:**
- `body` - Guild update data
- `reason` - Audit log reason

**Returns:** Promise resolving to the updated guild

**Example:**
```typescript
await guild.edit({
  name: 'New Guild Name',
  description: 'A cool community server'
}, 'Rebranding');
```
